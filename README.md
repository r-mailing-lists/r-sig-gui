# r-sig-gui

GUI Development

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-sig-gui/](https://stat.ethz.ch/pipermail/r-sig-gui/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics
